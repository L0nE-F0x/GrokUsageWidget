Download for Windows How to install

Free · Windows 10/11 · ~2 MB installer · Personal use

Live usage

Overall weekly percent, per-category bars, reset time, and extra usage credits.

Stays out of the way

Compact always-on-top window, hide to the system tray, refresh on a schedule or by hand.

Your account only

Sign in once in a private browser profile on your PC. Nothing is shared with a cloud service.

Install in a minute

  1. Download the Windows installer (Grok-Usage-Setup.exe).
  2. Run it and start Grok Usage from the Start Menu.
  3. Click Connect my Grok account, sign in when the browser opens, and wait for your numbers.
  4. First refresh can take a few minutes while helpers download. Later refreshes are much faster.

Windows SmartScreen: the app is not code-signed (personal project). If Windows says it “protected your PC,” choose More info → Run anyway.

Use your own grok.com account. Don’t share login folders between people.
